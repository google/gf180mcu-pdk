7.2 Model Syntax For MIM Capacitor
----------------------------------

**Input Parameters:**

- w: Top plate design width [m].

- l: Top plate design length [m].

**General Syntax:**

Xxx  node_1  node_2  cap_mim_model_name  w =  l=

**Examples:**

Xc1 1 2 cap_mim_1f0_m2m3_noshield w=50u l=50u
